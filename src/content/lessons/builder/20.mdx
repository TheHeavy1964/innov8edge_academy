# Multi-Agent Systems

When a task is too big for one brain, you build a **Squad**.

## The Power of Specialization
If you ask a single agent to "Build a Marketing Campaign," it has to do 100 different things. It will eventually lose track, hallucinate, or get overwhelmed.

In a **Multi-Agent System**, you assign specific "Roles" to different agents:
- **The Planner:** Breaks the big goal into small steps.
- **The Executor:** Performs the actual work.
- **The Critic:** Reviews the work for errors.

### Why Squads Outperform Individuals:
1. **Focus:** Each agent has a very small, specific system prompt.
2. **Quality Control:** Agents can check each other's work.
3. **Resilience:** If one agent fails, the orchestrator can retry just that step.

## Collaborative Intelligence
You are no longer just building a bot; you are building an **Organization of Intelligence**.
